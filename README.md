# BankingSystem
This project is a simple Banking System implemented in Java using Object-Oriented Programming (OOP) concepts. It simulates basic banking operations such as creating an account, depositing money, withdrawing money, and checking account details.

## Features
- Create a bank account with account number and holder name
- Deposit money into the account
- Withdraw money with balance validation
- View current balance
- Display complete account details

## Concepts Used
- Classes and Objects
- Encapsulation (private data members)
- Methods for operations
- User input using Scanner
- Menu-driven program

## How It Works
The program creates a `BankAccount` object and provides a menu-driven interface for performing different banking operations. It ensures proper validation, such as preventing withdrawal when the balance is insufficient.

## Technologies
- Java (Core Java)
